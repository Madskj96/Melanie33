# Melanie33

En interaktiv fødselsdagsgave, optimeret til mobil og automatisk publiceret med GitHub Pages.

Siden indeholder gaveåbning, animationer, lyd, billedkarruseller, trøjevalg, spillertryk, eget navn og nummer, størrelsesvalg samt Danske Mestre-mærket.

Den offentlige side publiceres på:

**https://madskj96.github.io/Melanie33/**
